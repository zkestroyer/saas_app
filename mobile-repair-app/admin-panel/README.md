# Admin Panel

> **Status:** Planned for V2

The Revivix admin panel will be a web-based dashboard for Super Admins to manage:
- Multi-tenant oversight
- User management across all tenants
- System-wide analytics and audit trail viewer
- Subscription and billing management

## Planned Technology
- Next.js + Supabase (shared database)
- Role: Super Admin only

## Current Admin Capabilities
Currently, tenant-level admin functions are available within the mobile app under the `(tenant)` route group. A dedicated web admin panel is on the V2 roadmap.
